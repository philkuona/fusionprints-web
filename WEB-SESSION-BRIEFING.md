## IMAGERY — GEMINI IS THE SOLE IMAGE SOURCE

This is a hard rule for the entire project, all pages, all time:
- Gemini generates ALL images — homepage, about, how it works, every future page
- No Unsplash, no Pexels, no CSS placeholders, no blank cards, no shape mockups
- If a section needs an image, Gemini makes it
- Images saved to public/images/ with descriptive names

### API key
Environment variable only — never hardcode, never commit to git:
os.environ["GEMINI_API_KEY"]

### How to generate
```python
import google.generativeai as genai
import base64, pathlib, os

genai.configure(api_key=os.environ["GEMINI_API_KEY"])
client = genai.Client()

response = client.models.generate_content(
    model="gemini-2.0-flash-preview-image-generation",
    contents=YOUR_PROMPT,
    config={"response_modalities": ["IMAGE", "TEXT"]}
)

for part in response.candidates[0].content.parts:
    if part.inline_data:
        img_data = base64.b64decode(part.inline_data.data)
        pathlib.Path("public/images/FILENAME.jpg").write_bytes(img_data)
        print("saved public/images/FILENAME.jpg")
```

### Gemini image brand rules — apply to every prompt
- Diversity non-negotiable: people of African descent prominently represented across the full image set
- No location identifiers: no landmarks, flags, street signs, recognisable skylines
- Warm natural light: golden hour tones, never cold or clinical
- Real moments: genuine emotion, not posed stock
- Premium interiors: clean, minimal, warm textures
- No text, watermarks, logos or UI elements inside any image
- No printer equipment or production machinery
- Print content when visible: abstract, landscape or nature — never sharp identifiable faces
- Style benchmark: Artifact Uprising, Mpix, Nations Photo Lab

### Quality checklist — every image must pass before saving
- Feels premium (would Mpix use this?)
- Diversity represented (African descent prominent)
- No location identifiers
- Lighting is warm
- No text or logos in image
- Correct dimensions for its use case
- Would not embarrass the brand if published
Fail any check — regenerate with refined prompt.

### Image naming
public/images/[section]-[description].jpg
Examples: card-prints-4x6.jpg, card-wall-16x20.jpg, card-finish-guide.jpg, about-lifestyle.jpg

### next/image — always, never raw img tags
Fill mode:
```tsx
<div className="relative h-64 w-full overflow-hidden rounded-xl">
  <Image src="/images/card-prints-4x6.jpg" alt="..." fill className="object-cover" />
</div>
```
